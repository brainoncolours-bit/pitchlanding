# Pitch Studio Landing Page

**"Where Founders Meet Their Future"** — A complete, production-ready landing page for Pitch Studio with 10+ sections, smooth navigation, and responsive design.

## 🚀 Quick Start

```bash
npm install      # Already done
npm run dev      # Start dev server (http://localhost:5173)
npm run build    # Production build
```

## ✨ What's Included

✅ **Complete 10-Section Landing Page**
- Hero, About, Problem, How It Works, Benefits
- Journey, Testimonials, FAQ, Final CTA, Navbar, Footer

✅ **Fully Structured**
- `pages/` - Home page combining all sections
- `components/common/` - Navbar & Footer
- `components/sections/` - 9 section components

✅ **React Router Setup**
- Smooth navigation between sections
- Mobile-responsive navbar with drawer menu
- Anchor-based section scrolling

✅ **Production Ready**
- Dark theme with Sky-400 accents
- Fully responsive (mobile, tablet, desktop)
- Tailwind CSS 4 optimized
- Zero build errors

## 🎨 Sections Built

1. **Hero** - "Where Founders Meet Their Future" with proof stats (500+ founders, 150+ investors, $50M+)
2. **About** - Pitch Studio as a movement with Delvin Danny founder story
3. **Problem** - 6-panel grid showing founder access gaps
4. **How It Works** - 5-step timeline with golden line connector
5. **Benefits** - Split-panel design (light for founders, dark for investors)
6. **Journey** - Bangalore & Kerala expansion cards + "Your City is Next"
7. **Testimonials** - 6 ecosystem voices (2 founders, 2 investors, 1 mentor, 1 student)
8. **FAQ** - 8 honest answers including investment guarantee question
9. **Final CTA** - "Stop Waiting for Permission" movement message
10. **Footer** - Navigation, social links, legal, and founder credit
